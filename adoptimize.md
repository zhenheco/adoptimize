## 1. 產品概述

### 1.1 願景

打造一個**半自動化廣告優化平台**，透過 AI 建議 + 人工確認 + 一鍵執行的模式，幫助中小企業和初階行銷人員大幅降低廣告管理的時間成本，同時提升廣告投放效率。

### 1.2 核心價值主張

| 價值 | 說明 |
|-----|------|
| **即時價值** | 免費健檢立即看到問題，無需學習即可獲得專業建議 |
| **降低門檻** | 不懂廣告也能透過系統建議優化，降低專業知識門檻 |
| **效率提升** | 減少 90% 手動檢查時間，一鍵執行優化建議 |
| **持續學習** | 系統持續監控並學習，建議越來越精準 |

### 1.3 商業模式

採用 **Freemium** 模式：

| 層級 | 功能 | 價格 |
|-----|------|------|
| **免費版** | 廣告健檢報告、問題診斷、基本儀表板 | $0 |
| **Pro 版** | 一鍵執行、自動監控、進階分析、多帳戶 | $49/月 |
| **Enterprise** | API 存取、自訂規則、專屬支援 | 聯繫銷售 |

### 1.4 產品定位

```
目標：減少 90% 的手動廣告管理時間

方式：
┌─────────────────────────────────────────────────────────┐
│  自動數據同步  →  規則引擎分析  →  產生建議  →  一鍵執行  │
│     (15分鐘)        (即時)         (自動)      (用戶確認) │
└─────────────────────────────────────────────────────────┘
```

### 5.2 疲勞度計算公式

```
疲勞度 = (CTR變化率權重 × 40%) + (投放頻率權重 × 30%) 
       + (投放天數權重 × 20%) + (轉換率變化權重 × 10%)
```

**各因子評分標準：**

| 因子 | 0-25 分 | 26-50 分 | 51-75 分 | 76-100 分 |
|-----|---------|---------|---------|----------|
| CTR 變化 | > 0% | 0% ~ -10% | -10% ~ -20% | < -20% |
| 頻率 | < 2 | 2-3 | 3-4 | > 4 |
| 天數 | < 7 天 | 7-14 天 | 14-30 天 | > 30 天 |
| 轉換率變化 | > 0% | 0% ~ -10% | -10% ~ -20% | < -20% |

### 5.3 疲勞警示門檻

| 狀態 | 分數範圍 | 建議行動 |
|-----|---------|---------|
| 🟢 健康 | 0 - 40 | 持續投放 |
| 🟡 注意 | 41 - 70 | 準備替換素材 |
| 🔴 疲勞 | 71 - 100 | 立即更換素材 |

| T-003 | 一鍵執行 | P0 | 確認後立即執行操作 |
| T-004 | 批次操作 | P1 | 同時處理多個建議 |
| T-005 | 操作歷史 | P1 | 記錄所有已執行操作 |
| T-006 | 忽略/稍後 | P1 | 暫時忽略某建議 |
| T-007 | 還原功能 | P2 | 還原已執行的操作 |

### 7.2 優先級計算邏輯

```
優先級分數 = 嚴重度基礎分 + 金額影響分 + 修復難度分 + 影響範圍分

- 嚴重度基礎分：Critical=100, High=70, Medium=40, Low=20
- 金額影響分：預估影響金額 / 100（上限 50 分）
- 修復難度分：一鍵=30, 簡單=20, 中等=10, 複雜=0
- 影響範圍分：每個受影響實體 +5 分
```

### 7.3 操作類型

| 操作 | 自動執行 | 確認要求 | 還原支援 |
|-----|---------|---------|---------|
| 暫停廣告/素材 | ✅ | 建議確認 | ✅ |
| 降低預算 | ✅ | 必須確認 | ✅ |
| 增加預算 | ✅ | 必須確認 | ✅ |
| 建立排除受眾 | ✅ | 建議確認 | ✅ |
| 結構調整 | ❌ | - | - |

## 9. 數據模型設計

### 9.1 核心實體關係

```
用戶 (User)
  │
  └── 廣告帳戶 (AdAccount)
        │
        ├── 廣告活動 (Campaign)
        │     │
        │     └── 廣告組 (AdSet)
        │           │
        │           └── 廣告 (Ad)
        │                 │
        │                 └── 素材 (Creative)
        │
        ├── 受眾 (Audience)
        │
        ├── 健檢報告 (HealthAudit)
        │     │
        │     └── 問題 (AuditIssue)
        │
        └── 建議 (Recommendation)
              │
              └── 操作歷史 (ActionHistory)
```

### 9.2 主要資料表

| 分類 | 資料表 | 用途 |
|-----|--------|------|
| 用戶與帳戶 | users, ad_accounts | 用戶認證、廣告帳戶連接 |

| 素材與受眾 | creatives, creative_metrics, audiences, audience_metrics | 素材與受眾管理 |
| 健檢與操作 | health_audits, audit_issues, action_history, recommendations | 健檢結果與操作記錄 |

## 附錄 A：術語表

| 術語 | 說明 |
|-----|------|
| CPA | Cost Per Acquisition，每次轉換成本 |
| ROAS | Return On Ad Spend，廣告投資報酬率 |
| CTR | Click Through Rate，點擊率 |
